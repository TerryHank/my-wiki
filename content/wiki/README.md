# 知识库文件夹结构

> 📁 `D:\MyWiki\content` 目录的中文说明

---

## 根目录

| 文件/文件夹 | 说明 |
|-----------|------|
| `purpose.md` | **项目目标** - 定义研究目标、关键问题、范围 |
| `schema.md` | **架构定义** - 6 种页面类型的规范和命名约定 |
| `raw/` | **原始资料区** - 未经加工的输入材料 |
| `wiki/` | **知识库** - 结构化、编译后的知识页面 |
| `.llm-wiki/` | **LLM 缓存** - 自动化工具的工作目录（勿手动修改） |

---

## 📁 raw/ - 原始资料输入区

### `raw/assets/`
- **资源文件** - 图片、音频、视频等媒体

### `raw/sources/` (583+ 文件)
- **原始资料库** - 会议记录、需求文档、技术规范、实验数据
- 保持原始状态，后续提炼到 `wiki/`

---

## 📁 wiki/ - 编译后的知识库

### 6 种页面类型

| 目录 | 类型 | 数量 | 说明 |
|------|------|------|------|
| `wiki/entities/` | **实体** | 113 | 具体的人、事、物（工具、数据集、项目、服务器） |
| `wiki/concepts/` | **概念** | 78 | 抽象概念（技术、方法、框架、理论） |
| `wiki/sources/` | **资料** | - | 外部文献、论文、教程、文档 |
| `wiki/queries/` | **问题** | - | 开放性问题、正在调查的研究课题 |
| `wiki/comparisons/` | **对比** | - | 技术/工具/方案的横向对比分析 |
| `wiki/synthesis/` | **综合** | - | 跨来源的综合结论和最终方案 |

### 索引文件

| 文件 | 说明 |
|------|------|
| `wiki/index.md` | **简洁索引** - 按类型列出所有页面 |
| `wiki/log.md` | **研究日志** - 按时间记录构建活动 |
| `wiki/overview.md` | **项目概述** - 知识库整体介绍 |

---

## 🔗 知识连接

- 使用 `[[双向链接]]` 语法
- 不依赖中央索引，让知识自然连接
- 通过图谱视图探索概念关系

---

## 🌱 数字花园理念

> 让知识像植物一样自然生长、相互缠绕，而不是僵化的层级结构。

- ✅ 持续生长：内容持续演化的思考记录
- ✅ 自然连接：双向链接形成知识网络
- ✅ 渐进完善：原始资料 → 结构化知识 → 综合结论
- ✅ 图谱探索：可视化概念间的联系

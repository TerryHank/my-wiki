# BIM 算量系统

本文档描述了从 2D CAD 图纸自动提取材料用量的系统规划。

## 核心目标
旨在通过自动化流程替代人工统计，从工程图纸中精确提取管线长度、电缆路由等数据，最终生成准确的材料采购清单。

## 技术管线
系统遵循以下数据处理流程：
1.  **DWG → DXF**：将原生 DWG 格式转换为便于解析的 DXF 交换格式。
2.  **解析**：读取 DXF 文件中的几何图元。
3.  **拓扑分析**：从离散的 2D 线条中重建逻辑连接关系。
4.  **管线续接**：将断开的线段逻辑连接为连续的管线对象。
5.  **采购清单**：基于重建的模型统计工程量。

## 关键挑战
- **2D 管线续接准确性**：如何从非智能的 2D 线条中正确推断工程逻辑（如交叉与连通的区别）。
- **字符编码处理**：解决 CAD 文件中中文标注的编码混乱问题。
- **多图纸交叉引用**：处理系统图与平面图分散在不同文件中的关联信息。

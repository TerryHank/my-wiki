# Microsoft

Microsoft Corporation is a multinational technology organization heavily involved in artificial intelligence research and cloud computing.

## Role in Computer Vision

Microsoft played a pivotal role in the computer vision community by leading the creation of the **[[coco-dataset]]** (Common Objects in Context). The dataset was developed by researchers including Tsung-Yi Lin during their time at Microsoft Research. It has since become the industry standard for benchmarking object detection and segmentation models.

## Cloud Infrastructure

In the context of this wiki, Microsoft is also the provider of **[[azure-ml]]** (Azure Machine Learning), a cloud platform used for remote training of deep learning models, offering an alternative to [[aws-ec2]] for scaling compute resources.

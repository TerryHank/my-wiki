# BIM 算量系统

**BIM 算量系统**是一个旨在实现从 2D 工程图纸到材料采购清单全自动转化的软件项目。尽管名称包含"BIM"，其当前技术路线主要基于对 2D CAD/DWG 文件的深度解析与拓扑重建。

## 核心功能
- **自动提取**：从图纸中识别并提取管线长度、电缆路由等关键几何信息。
- **逻辑重建**：通过算法将离散的 2D 线段重组为具有工程意义的连续管线。
- **统计输出**：生成精确的材料用量表，服务于采购与成本控制。

## 技术架构
系统依赖以下关键模块：
1.  **格式转换**：利用工具将 DWG 转换为 DXF。
2.  **数据解析**：处理 DXF 文件流，提取图元数据。
3.  **[[2d-topology-reconstruction]]**：核心算法模块，负责分析线条间的空间关系。
4.  **管线续接模块**：解决 2D 绘图中常见的断线问题，建立逻辑连通性。

## 开发基础设施
- 主要开发与运行环境为 [[labpc-workstation]]。
- 遵循 [[skill-file-architecture]] 模式，相关操作指南存储于 `raw/articles/` 目录下。

## 挑战与局限
- **2D 语义歧义**：从 2D 线条反推 3D 逻辑存在天然困难，需处理大量特殊情况（如视觉交叉但逻辑断开）。
- **编码问题**：需妥善处理中文字符编码以避免解析错误。
- **多文件关联**：需建立机制处理分散在多张图纸中的系统信息。

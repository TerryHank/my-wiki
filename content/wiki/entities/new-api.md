# New-API

**New-API** 是一个通过 Docker 容器部署的后端 API 服务项目。

## 特点
- **标准化交付**：利用 Docker 技术栈简化依赖管理和部署流程。
- **运维集成**：纳入 [[codex-cli]] 的自动化运维范畴，支持快速迭代和重启。

## 关联
- 与 [[aiclient2api]] 类似，属于系统后端核心服务组件。

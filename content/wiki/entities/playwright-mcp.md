# Playwright MCP

**Playwright MCP** 是一个基于 Model Context Protocol (MCP) 的工具集成，将 Playwright 浏览器自动化能力赋予 AI 代理。

## 功能
- 允许 [[codex-cli]] 直接控制浏览器执行网页交互、测试和数据提取任务。
- 将图形化界面的操作转化为 AI 可理解的协议调用。

## 安装与配置
- 通过 Codex 会话执行安装命令进行集成。
- 是 **MCP 工具生态扩展** 的关键组成部分，标志着 AI 代理从文件操作向网络交互的演进。

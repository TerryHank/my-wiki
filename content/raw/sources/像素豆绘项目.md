# 像素豆绘 (PixelDoodle)

ESP32 像素艺术设备 + Taro 多端前端 + BLE 通信。

## 项目结构
- `firmware/` — ESP32 固件（Arduino）
- `frontend-taro/` — Taro React + TypeScript 多端前端
- `docs/` — 规格文档

## 技术栈
- **固件**: ESP32 + Arduino + BLE
- **前端**: Taro (React + TypeScript + Sass)
- **构建**: H5 + 微信小程序 + React Native
- **状态管理**: Zustand
- **通信**: BLE (Web Bluetooth / 小程序蓝牙 API)

## Codex 任务线程
- Taro 工程初始化（任务 1-5，子代理驱动开发）
- RN Android 扩展
- 规格审查 + 代码质量审查（多轮）
- 固件引脚修改、UUID 二维码功能

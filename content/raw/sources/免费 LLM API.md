# FreeLLMAPI

统一 LLM API 代理平台（类似 sub2api）。

## 功能
- 统一 API 密钥管理
- 回退链（Fallback Chain）
- 对话测试 Playground
- 数据分析 Analytics
- 供应商密钥管理

## Codex 工作
- 中文化所有页面（Analytics, Fallback, Playground, Keys）
- Docker 部署
- Response2Chat 兼容层（Mimo 适配）

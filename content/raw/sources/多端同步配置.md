# 多端同步配置

通过 GitHub 私有仓库 + obsidian-git 插件实现多设备同步。

## GitHub 仓库
- **地址**: https://github.com/TerryHank/MyWiki (私有)
- **代理**: gh-proxy.com（国内加速）

## 当前设备 (TerryLaptop)
- **路径**: D:\MyWiki
- **同步方式**: obsidian-git 插件自动同步（每 10 分钟）
- **Remote**: `https://gh-proxy.com/https://github.com/TerryHank/MyWiki.git`

## 新设备克隆步骤

### Windows
```bash
# 1. 安装 Obsidian
# 2. 克隆仓库（用 gh-proxy 代理）
git clone https://gh-proxy.com/https://github.com/TerryHank/MyWiki.git D:\MyWiki

# 3. 用 Obsidian 打开 D:\MyWiki
# 4. 安装社区插件: obsidian-git, dataview, hermes-console
# 5. obsidian-git 会自动识别 .git 配置
```

### Linux / WSL
```bash
# 克隆
git clone https://gh-proxy.com/https://github.com/TerryHank/MyWiki.git ~/MyWiki

# 安装 Obsidian 或用 Hermes Agent 操作
```

### LabPC (10.0.0.2)
```bash
git clone https://gh-proxy.com/https://github.com/TerryHank/MyWiki.git C:\Users\Virtual\Documents\MyWiki
```

## 同步规则
- **自动拉取**: 每 10 分钟
- **自动推送**: 每 10 分钟（有变更时）
- **冲突处理**: merge 策略
- **提交信息**: `vault backup: {date}`

## 手动同步
```bash
cd /d/MyWiki
git pull origin main    # 拉取
git add -A && git commit -m "手动同步" && git push  # 推送

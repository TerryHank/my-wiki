# LabPC BIM/CAD 算量项目

## 项目目标

从 2D CAD/DWG 工程图纸自动提取材料数量（算量）

## 用途

**采购**: 提取管道长度、线路等，用于材料采购

## 技术流程

```
DWG 图纸 → DXF 转换 → 解析 → 拓扑分析 → 采购清单
```

### 步骤说明

1. **DWG → DXF 转换**:
   - 使用 CAD 软件或库（如 ezdxf）
   - DXF 为开放格式，更易解析

2. **DXF 解析**:
   - 提取几何信息（线、圆、弧等）
   - 提取图层信息
   - 提取属性数据

3. **拓扑分析**:
   - 识别连接关系
   - 计算长度、面积
   - 分类材料类型

4. **采购清单生成**:
   - 按类型分组
   - 计算数量
   - 导出格式（Excel/CSV）

## Hermes 技能

### 技能名称

**bim-quantity-takeoff**

### 功能

- DWG→DXF 转换
- DXF 解析
- 拓扑分析
- 采购清单生成

### 应用领域

- MEP（机械、电气、管道）
- 电气工程
- 结构工程

## 技术栈

### Python 库

- **ezdxf**: DXF 读写
- **shapely**: 几何计算
- **pandas**: 数据处理
- **openpyxl**: Excel 导出

### CAD 软件

- AutoCAD（商业）
- LibreCAD（开源）
- 在线转换工具

## 输入输出

### 输入

- DWG 图纸文件
- 图纸标注
- 材料规格

### 输出

- 材料清单（BOM）
- 数量统计
- 成本估算（可选）

- 弯头数量
- 阀门数量

### 2. 电气工程

提取:
- 线缆长度
- 接线盒数量
- 开关数量
- 插座数量

### 3. 暖通空调

提取:
- 风管长度
- 风口数量
- 设备清单

## 优势

- **自动化**: 减少人工计算错误
- **快速**: 大幅提升效率
- **准确**: 统一计算规则
- **可追溯**: 来源清晰

## 挑战

### 1. 图纸标准化

不同图纸标准差异大，需要适应多种格式

### 2. 语义理解

理解图纸标注和符号，需要领域知识

### 3. 复杂几何

不规则形状、斜线、曲线等计算复杂

### 4. 多图纸集成

跨图纸引用、总图与详图关联
